🎓 College Portfolio Website

A full-stack portfolio website project developed as part of college work.
It showcases personal projects, skills, and includes a backend system to store user/contact data using PostgreSQL.

🚀 Features
👤 Personal portfolio showcase
📂 Projects section
📞 Contact form (stores data in database)
🌐 Backend API using Node.js
🗄️ PostgreSQL database integration
☁️ Hosted on Render
🔄 Continuous Integration (CI) with GitHub Actions


🛠️ Tech Stack
Frontend
HTML
CSS (Glassmorphism UI)
JavaScript
Backend
Node.js
Express.js
PostgreSQL (pg)
Deployment & Tools
Render
